# Notes:
# - Modern ale-py (0.9+) bundles Atari ROMs directly; `AutoROM --accept-license`
#   is a safety-net no-op in that case, and a required one-time download on older ale-py.
# - RAM observations use gym.make("ALE/<Game>-v5", obs_type="ram"), the officially
#   documented mechanism, so this code works unchanged across ale-py/gymnasium versions.
# - Do NOT pin numpy<2.0: Colab's base image ships numpy>=2.0 and many pre-installed
#   packages (opencv, jax, tifffile, etc.) are built against its ABI. Downgrading numpy
#   breaks all of them with "numpy.dtype size changed" errors. The old-gym deprecation
#   warning about NumPy 2.0 is harmless and safe to ignore - leave numpy unpinned.
